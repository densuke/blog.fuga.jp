授業でvagrant+saharaによるロールバック機能を試させることにしました。

    $ mkdir -pv ~/nw/db
    $ cd ~/nw/db
    $ vagrant init ubuntu/trusty64
    $ vagrant up
    ...
    $ vagrant sandbox on
    $ vagrant ssh
    # VMの中で "rm -fr" する
    # 安全のため、特殊スイッチをつけないとやってくれません
    VM$ sudo rm -fr --no-preserve-root /  
    $ exit
    $ vagrant sandbox rollback

とするとロールバックに失敗します。
「お」「おおぉぉぉぉぉぉぉ…」となってしまいました。

理由を考えながら `ls` するとわかりました…

**/vagrant越しにVagrantfileや管理ディレクトリ(.vagrant)も消していた** のです。

ということで、やっぱり **ダメ、ゼッタイ!** なのでした orz
