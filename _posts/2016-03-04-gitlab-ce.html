  • GitLab 8.5.3 Released | GitLab
現在8.5.1を使っているのでそのまま更新可能なはずですが、どういうわけか、自宅のサーバーでは素直にapt-get upgradeでgitlab-ceを更新しようとするとI/Oが応答しなくなることがあるため、一度リブートしてから

$ sudo dpkg --configure -a$ sudo apt-get install -f #一度インストール指示を出してるので残りをしてくれそう

となってます。ちょっと今回は手法を変えて、

$ sudo apt-get remove gitlab-ce$ sudo apt-get install gitleb-ce

として、最後に

$ sudo gitlab-ctl reconfigure ; sudo gitlab-ctl restart

で対応してみてます。あ、うまくいったかな?
